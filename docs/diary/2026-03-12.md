## Q&A Log

**Q: Xcode build failed. Debug.**
A: `xcode-select` was pointing to Command Line Tools instead of Xcode.app. Fixed by running `sudo xcode-select -s /Applications/Xcode.app/Contents/Developer`. Then ran `npm run build` + `npx cap sync ios` to rebuild web assets and sync to iOS before re-opening in Xcode.

**Q: White space below, profile icon way too high, screen should fit the phone.**
A: Multiple iterations:
1. Changed outer div from `min-h-screen` â†’ `h-screen`, inner div to `h-full overflow-y-auto` with inline safe area padding.

**Q: While login, failed to fetch issue.**
A: CORS issue. Backend only allowed `http://localhost:5173`, but dev server started on `5174` (port 5173 was held by a stale node process). Fixed by killing the stale process (`kill <pid>`) and restarting the dev server, which then claimed port 5173.
